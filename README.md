# AetherBlade

##  О проекте

**AetherBlade** — это 2D action roguelite в пиксель-арт стилистике, разрабатываемый на движке Unity. Игра сочетает динамичный ближний бой с системой парирования и накопительными усилениями.

Игрок управляет **Лексом**, стажёром корпорации по добыче Итрия — источника невероятной энергии. Пережив техногенный апокалипсис, герой использует клинок из чистой энергии, чтобы выжить и докопаться до истины.


##  Основные механики

###  Бой
*   **Атака** - основной способ нанесения урона врагам.
*   **Парирование** - своевременное блокирование вражеских атак, открывающее окно для контратаки.
*   **Шкала усиления** - заполняется в бою при успешных атаках и парированиях. При активации даёт кратковременное огромное усиление урона.

###  Здоровье и урон
*   У игрока и врагов есть полоски здоровья.
*   Получение урона сопровождается визуальными и звуковыми эффектами.

###  Система улучшений 
После каждой попытки игрок получает ресурсы для постоянного прогресса:

*   **Улучшение здоровья** - увеличение максимального запаса прочности.
*   **Улучшение урона** - повышение базового урона клинка.
*   **Улучшение скорости восстановления шкалы усиления** - быстрый доступ к усиленному состоянию.



##  Враги

В вертикальном срезе реализовано **2 типа врагов** с разными паттернами поведения и атак.

##  Управление

| Кнопка | Действие |
| :--- | :--- |
| `W` `A` `S` `D` | Передвижение игрока |
| `Левая кнопка мыши` | Атака |
| `ESC` | Меню |

##  Атмосфера и подача

*    **1 фоновый трек**, поддерживающий атмосферу.
*    **Звуковые эффекты** для действий в игре (атака, парирование, получение урона, активация усиления).

##  Технологии

*   **Unity** - игровой движок.
*   **C#** - язык программирования.
*   **GitHub** - система контроля версий.
*   **Aseprite / Piskel** - инструменты для создания пиксель-графики.
